# LegacyTextField

<Information
  text='deprecated'
  description='**Denne komponenten er erstattet av ny [Textfield](/?path=/docs/komponenter-textfield--docs) komponent.**'
/>

`TextField` er en tekstboks som tar imot korte, rene tekster uten linjeskift. For lengre tekster, bruk `TextArea`.

## Bruk

```tsx
import { LegacyTextField } from '@digdir/design-system-react';

<LegacyTextField {...props} />;
```
