# List

Liste brukes for å vise en liste med elementer. Listen kan inneholde både tekst, linker og ikoner.

<Primary />
<Controls />

## Slik bruker du `List`

Lista kan være både ordnet og uordnet, dette endrer du ved å bruke `List.Ordered` eller `List.Unordered`.

`List.Root` er påkrevd for å får riktig aria attributter (`aria-labelledby`) mellom listen og tittel.

```tsx
import '@digdir/designsystemet-theme'; // Importeres kun en gang i appen din.
import { List } from '@digdir/designsystemet-react';

<List.Root>
  <List.Heading>Heading</List.Heading>
  <List.Ordered>
    <List.Item>Item</List.Item>
  </List.Ordered>
</List.Root>;
```

## Eksempler på bruk av `List`

### Uten heading

Dersom du bruker `List.Heading` så legger vi automatisk på `aria-labelledby` på `List.Heading` og `List.Ordered/Unordered`
for å gi skjermlesere beskjed om at `List.Heading` er en heading for `List`.

Dersom du ikke legger til en heading kommer vi ikke til å legge til `aria-labelledby`.

<Canvas of={ListStories.WithoutHeading} />

### Avansert

Du kan nøste lister så mye du vil, og stylingen vil tilpasse seg automatisk. Dette er browserens default styling.

### List of links
